マットな質感のフロストガラスでお菓子のカヌレを表現

お菓子のカヌレを逆さにしたような愛らしい姿をマットな質感のフロストガラスで表現しました。アルマイト加工を施した丸みのあるシルバーカラーのソケットカバーが、フレンチシックなムードを漂わせます。玄関やレストルームなどの小さなスペースをやさしく彩ります。

- Features -
Function 01
陰影

なだらかな凹凸を持たせたフロストガラスは点灯させると繊細な陰影を描き出します。
Function 02
アルマイト加工

ソケットカバーにはアルミのような軽やかな質感が特徴のアルマイト加工を施しています。
Function 03
華やぎ

膨らみや曲線で表現した装飾的なデザインが小さなスペースに華やぎをもたらします。
Function 04
オブジェクト

マットな質感のフロストガラスは点灯時も消灯時も品の有るオブジェクトとして空間を彩ります。
Function 05
サイズ感

デザインを損なわず実用性を両立させた、ベストなサイズ感を追求。

「人が過ごす場所と時間を心地よく」

インターフォルムは1979年に照明器具のメーカーとして創業して以来、時計やインテリア雑貨にもフィールドを広げ、
みなさまの日々の暮らしのひとときを心地よくするお手伝いをしたいという想いを込めて、オリジナルデザインの商品をご提案してきました。

インターフォルムという社名には、「世界に通用するデザインを」という意味が込められています。
そんな理想を持ち、世界のトレンドと、日本の緻密なデザインワークをかけ合わせた商品づくりを続けています。
