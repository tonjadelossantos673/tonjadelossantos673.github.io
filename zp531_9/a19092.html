- Features -
Function 01
螺旋

ガラスの内側に螺旋状の模様を閉じ込めました。凹凸が生む繊細な陰影が優美なムードを漂わせます。
Function 02
ゴールド

ソケットは真鍮を思わせる落ち着きのあるゴールドカラーで。華やかな印象を与えます。
Function 03
真球

ガラスとソケットを真球フォルムになるようデザインしています。ミニマルなシルエットがガラスの模様を一層引き立てます。
Function 04
存在感

ビー玉や万華鏡のような奥行きのある陰影は、消灯させていても、その存在感を発揮します。
Function 05
サイズ感

デザインを損なわず実用性を両立させた、ベストなサイズ感を追求。

「人が過ごす場所と時間を心地よく」

インターフォルムは1979年に照明器具のメーカーとして創業して以来、時計やインテリア雑貨にもフィールドを広げ、
みなさまの日々の暮らしのひとときを心地よくするお手伝いをしたいという想いを込めて、オリジナルデザインの商品をご提案してきました。

インターフォルムという社名には、「世界に通用するデザインを」という意味が込められています。
そんな理想を持ち、世界のトレンドと、日本の緻密なデザインワークをかけ合わせた商品づくりを続けています。
